# JavaLottery

目的: 之前在C#_WindowsForms有做過小型樂透,所以試著嘗試用其它程式語言,像是Java

小型樂透(C#_WindowsForms): https://github.com/TinaLebron/Csharp_WindowsFormsSmallWork

功能: <br/>
• 部分選號: 先選幾個數字,在選擇部分選號,如果沒有選任何數字或已超過數量,則會有提示框<br/>
• 全部快選: 必須都沒有選擇數字前提下,不然會有視窗提示<br/>
• 清除選號: 把選過的數字清空<br/>
• 加入號碼組: 選擇6個數字後,後續要跟中獎號碼做比對<br/>
• 刪除號碼組: 清除號碼組<br/>
• 開獎: 有7個數字,依照中獎方式判別中獎結果(會有視窗顯示)<br/>


改善: <br/>
• 刪除號碼組可增加刪除多筆的功能<br/>

| Option | Description |
| ------ | ----------- |
| Partial selection(部分選號)  | First select a few numbers, select some of the selection numbers. |
| All quick selection(全部快選)  | There must be no choice of numbers, otherwise there will be a window prompt. |
| Clear selection(清除選號)  | Empty the selected number. |
| Join a number group(加入號碼組)  | After selecting 6 numbers, follow up with the winning number. |
| Delete number group(刪除號碼組)  | Clear number group. |
|Lottery(開獎)  | There are 7 numbers, and the winning result is judged according to the winning method (there will be a window display). |
